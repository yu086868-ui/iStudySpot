# 卡片发布与收集系统

卡片系统作为微信小程序的拓展功能，与主功能保持相对独立，用于根据学习行为生成并收集AI卡片。



## 功能边界

1. AI生成卡片文本
2. AI生成卡片图片
3. md解析与卡片显示
4. 服务器生成卡片数据
5. 收集展示页面
6. 稀有度玩法



## 卡片数据结构

服务器：

- UUID
- 用户ID
- cardID、

AI -> 服务器：

- md文本
- image (imageURL）



## AI协议

### AI文本prompt

```
AI返回Markdown。

限制：
- 一个一级标题
- 使用 --- 作为卡片分块
- 不使用复杂md结构
```

解析规则

```
# → 卡片标题

--- → 分卡片

> → 引用样式

正文 → 普通文本
```



### AI图片prompt

```
像素512*512
```
