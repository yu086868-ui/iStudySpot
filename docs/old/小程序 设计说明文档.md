# 设计说明

## 配色方案

- 主色：#4CAF7A（用于核心按钮、选中状态、强调操作）
- 辅助色：#E8F5EE（用于卡片背景、弱强调区域）
- 强调色：#3E9B68（用于主色的深色态，如点击/悬停）

> 补充说明（可写可不写）：
>  背景色 #F4F7F5，文本主色 #1F2933，次级文本 #6B7280，用于保证整体可读性与层级

------

## 字体

- 标题：PingFang SC（中文） / Inter（英文），Semibold，20~24px
- 正文：PingFang SC / Inter，Regular，14px

------

## 核心页面

1. 首页 - 核心入口页面，提供“选座/开始学习”入口及信息卡片展示
2. 选座页面 - 展示座位状态（可选/已占用/已选中），完成选座操作
3. 学习状态页 - 展示当前学习状态，提供签到/签退功能（沉浸式页面）
4. 规则公告页面 - 展示自习室规则与公告信息
5. 个人中心页面 - 用户信息管理、当前预约查看等功能

------

## 交互说明

- 点击首页“选座” → 跳转到选座页面
- 在选座页面点击座位 → 选中座位 → 点击确认进入学习状态页
- 在学习状态页点击“签退” → 结束学习状态
- 底部导航点击 → 在“规则公告 / 首页 / 个人中心”之间切换
- 页面支持下拉刷新（用于更新座位状态/公告信息）
