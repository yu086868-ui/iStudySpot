### 前端（微信小程序）

技术栈：
- 微信小程序原生框架
- WXML + WXSS + TypeScript

运行环境：
- 微信小程序开发者工具

通信方式：
- 使用 wx.request 与后端进行 RESTful API 通信

选择理由：
1. 微信小程序原生框架与微信生态高度兼容，适合快速开发和调试
2. 使用 TypeScript 提供类型检查，提高代码安全性和可维护性
3. 原生框架结构清晰，支持页面与组件分离，便于模块化开发
4. 小程序 API 完整，能够满足课程项目的功能需求

补充说明：
当前项目基于官方 TypeScript 模板初始化，具备完整类型定义（miniprogram-api-typings），符合工程化开发要求
