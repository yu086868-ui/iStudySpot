### 前端开发环境

开发工具：
- 微信开发者工具（用于小程序开发、调试与预览）
- Node.js（用于依赖管理）

项目初始化：
- 使用微信小程序 TypeScript 模板初始化项目
- 自动生成基础目录结构及配置文件

关键配置说明：

1. project.config.json
- 指定小程序根目录为 miniprogram/
- 配置项目 AppID
- 启用 TypeScript 编译支持

2. tsconfig.json
- 开启严格模式（strict）
- 目标版本：ES2020
- 提供类型检查与编译规则

3. package.json
- 引入 miniprogram-api-typings
- 提供微信小程序 API 的类型定义支持

4. app.json
- 配置全局页面路径
- 设置窗口样式（导航栏、标题等）
- 启用组件懒加载机制
