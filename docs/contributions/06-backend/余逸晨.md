## 我完成的工作

### 1. 后端项目架构

基于 Spring Boot 3.x + MyBatis 构建后端服务，采用分层架构：

```
com.ycyu.istudyspotbackend
├── controller/          # 控制器层（REST API）
├── service/             # 服务层（业务逻辑）
├── mapper/              # 数据访问层（MyBatis Mapper）
├── entity/              # 实体类
├── dto/                 # 数据传输对象
├── config/              # 配置类
├── interceptor/         # 拦截器
├── utils/               # 工具类
└── agent/               # AI Agent 模块
```

## 遇到的问题和解决

### 1. 问题：DTO 字段映射冲突
- **描述**：原有 Order 实体使用 @JsonProperty 注解，导致微信小程序接口响应字段名不一致
- **解决**：创建独立的 WxOrderDTO，在 DTO 层完成字段映射，不修改原有实体

### 2. 问题：头像上传路径问题
- **描述**：头像保存到绝对路径，其他电脑无法访问
- **解决**：改为基于项目运行目录的相对路径（uploads/avatars/）

### 3. 问题：卡片图片 401 错误
- **描述**：图片路径 /api/card/image/** 未加入 JWT 白名单
- **解决**：在 WebConfig 的 excludePathPatterns 中添加 "/api/card/image/**"

### 4. 问题：预约接口 500 错误
- **描述**：order 表缺少 total_price、updated_at 字段
- **解决**：执行 SQL 添加缺失字段

### 5. 问题：规则接口无数据
- **描述**：findGeneralRules() 查询 study_room_id IS NULL 的规则，但数据中均有值
- **解决**：修改为 findAll()，返回所有规则
