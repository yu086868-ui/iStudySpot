# UI / UX 设计说明

本应用采用 **底部导航（Bottom Navigation）** 作为主要导航方式，用于在四个核心功能页面之间进行切换。

主要页面包括：

- 首页
- 规则介绍
- 更多功能
- 用户中心

导航栏固定在页面底部，包含四个入口，用于在各页面之间快速切换。需要显示导航栏的页面将在具体实现阶段另行说明。

------

# 一、首页（Home）

首页用于展示应用的主要功能入口，并引导用户进入核心业务流程。

页面整体为 **纵向结构布局**，自上而下分为三个主要区域。

## 1. 常用功能区

页面顶部为常用功能入口区域。

设计特点：

- 以 **网格形式**展示功能入口
- 每行展示 **4 个功能入口**
- 每个入口包含：
  - 功能图标
  - 功能名称

这些功能属于应用中的 **高频使用功能**，从“更多功能”模块中提取出来，以便用户快速访问。

该区域需要具备一定的 **扩展性**，后续可根据用户使用情况调整或增加入口。

------

## 2. 核心功能入口

页面中部设置一个 **主要操作入口区域**，用于引导用户进入应用的核心流程。

该区域通过一个视觉明显的 **图片或卡片式入口** 展示。

交互逻辑：

- 用户点击该区域后进入 **选座界面**
- 该界面为应用的核心操作页面

视觉设计建议：

- 入口区域尺寸较大
- 具有明显的点击提示
- 可以配合简短的说明文字

------

## 3. 底部导航栏

页面底部为全局导航栏，用于在主要功能页面之间进行切换。

------

# 二、规则介绍（Rules）

规则介绍页面用于向用户说明应用的使用规则以及常见问题。

页面整体以 **信息展示为主**，采用清晰的上下结构。

页面分为两个主要部分。

------

## 1. 规则说明

页面上半部分用于展示应用的主要使用规则，例如：

- 使用规范
- 预约规则
- 违规处理方式
- 其他注意事项

内容以 **标题 + 说明文字** 的形式展示，必要时可使用列表或分点说明以提高可读性。

------

## 2. 常见问题（FAQ）

页面下半部分展示常见问题及解答，例如：

- 如何预约座位
- 如何取消预约
- 违规行为如何处理
- 忘记签到怎么办

建议采用以下展示方式之一：

- 折叠式问题列表
- 问题 + 答案列表

以减少页面信息密度，提高阅读体验。

------

## 3. 底部导航栏

页面底部显示全局导航栏。

------

# 三、更多功能（More）

更多功能页面用于集中展示应用中的 **所有辅助功能入口**。

该页面相当于应用的 **功能中心**，用于补充首页未展示的功能。

------

## 1. 功能分类

为了提高可读性，功能按类别进行分组展示，例如：

**预约相关**

- 预约记录
- 我的座位
- 取消预约

**账户相关**

- 充值
- 钱包记录
- 优惠活动

**系统功能**

- 意见反馈
- 帮助中心
- 关于应用

------

## 2. 功能入口展示

每个分类下以 **网格形式**展示功能入口。

每个入口包含：

- 功能图标
- 功能名称

这种布局可以在保持页面整洁的同时，提高用户查找功能的效率。

------

## 3. 设计目标

该页面的设计目标包括：

- 集中展示应用所有功能
- 避免首页信息过于拥挤
- 为未来功能扩展预留空间

------

## 4. 底部导航栏

页面底部显示全局导航栏。

------

# 四、用户中心（User Center）

用户中心用于展示用户个人信息以及与账户相关的功能。

整体结构分为 **用户信息区域** 和 **功能入口区域**。

------

## 1. 用户信息区域

页面顶部展示用户基本信息，包括：

- 用户头像
- 用户名称

该区域用于突出用户身份，并作为个人页面的视觉中心。

------

## 2. 个人功能入口

用户信息下方为个人相关功能入口，采用 **列表式结构** 展示。

包含以下功能：

- 消息中心
- 我的钱包
- 个人设置
- 退出登录

每个功能入口包含：

- 功能名称
- 功能图标
- 进入指示（例如箭头）

用户点击后进入对应的功能页面。

------

## 3. 底部导航栏

页面底部显示全局导航栏。

------

# 五、导航结构

应用整体采用 **底部导航结构** 作为主导航方式。

导航入口包括：

| 页面     | 功能                   |
| -------- | ---------------------- |
| 首页     | 展示常用功能与核心入口 |
| 规则介绍 | 展示规则说明与常见问题 |
| 更多功能 | 展示全部功能入口       |
| 用户中心 | 展示用户信息与账户功能 |

底部导航在页面之间提供快速切换能力，并保持统一的导航体验。
